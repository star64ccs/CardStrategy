# 🎴 CardStrategy 組件庫

歡迎使用 CardStrategy 組件庫！這是一個專為卡片策略應用設計的完整組件系統，包含豐富的動畫效果和專業的用戶界面。

## 🚀 核心特色

### 高級動畫系統
- **60fps 流暢動畫**: 所有動畫都經過優化，確保流暢的用戶體驗
- **豐富動畫效果**: 3D旋轉、波浪、脈衝光環、粒子爆炸等
- **性能優化**: 動畫緩存、批量處理、節流控制
- **統一配置**: 統一的動畫時長和緩動函數

### 專業設計系統
- **深色主題**: 現代化的深色主題設計
- **金色元素**: 優雅的金色強調色
- **響應式設計**: 適配不同設備和屏幕尺寸
- **一致性**: 統一的設計語言和視覺風格

### 完整組件庫
- **基礎組件**: Button、Card、Input、Badge、Avatar
- **動畫組件**: AnimatedCard、AnimatedButton、AnimatedView
- **專業組件**: CardGradingDisplay、CardPriceChart、AIRecommendationCard
- **交互組件**: Modal、Switch、Tooltip、LoadingSpinner

## 🎯 使用指南

### 快速開始

```tsx
import { Button, Card, Badge } from '@components/common';

// 基礎使用
<Button onPress={() => console.log('Pressed!')}>
  點擊我
</Button>

// 動畫卡片
<AnimatedCard animationType="slideUp" title="動畫卡片">
  內容
</AnimatedCard>

// 專業組件
<CardGradingDisplay result={gradingResult} />
```

### 主題配置

```tsx
import { theme } from '@theme/designSystem';

// 使用主題顏色
const styles = StyleSheet.create({
  container: {
    backgroundColor: theme.colors.background.primary,
    borderColor: theme.colors.gold.primary,
  }
});
```

## 📱 組件分類

### 基礎組件
- **Button**: 多種變體的按鈕組件
- **Card**: 基礎卡片容器
- **Input**: 輸入框組件
- **Badge**: 徽章和標籤
- **Avatar**: 頭像組件

### 動畫組件
- **AnimatedCard**: 高級動畫卡片
- **AnimatedButton**: 動畫按鈕
- **AnimatedView**: 動畫視圖容器
- **LoadingSpinner**: 加載動畫
- **ProgressBar**: 進度條動畫

### 專業組件
- **CardGradingDisplay**: 卡片評級顯示
- **CardPriceChart**: 價格圖表
- **CardCollectionManager**: 收藏管理
- **AIRecommendationCard**: AI推薦卡片
- **CardScannerOverlay**: 掃描覆蓋層

### 交互組件
- **Modal**: 模態對話框
- **Switch**: 開關組件
- **Tooltip**: 工具提示
- **Toast**: 提示消息
- **Skeleton**: 骨架屏
